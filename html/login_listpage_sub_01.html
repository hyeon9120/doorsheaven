  • 작성일

  • 조회수

  • 0001

1. 일시 : 2019년 3월 27일 (수) 오전 9시
2. 장소 : 충북 청주시 홍덕구 오송읍 오송생명6로 71 (주)메디톡스 오송공장 2층
3. 회의의 목적사항
가) 보고사항 : (1) 감사보고 (2)영업보고 (3) 내부회계관리제도 운영실태 보고 나) 부의안건
제1호 의안 : 제19기 재무제표 승인의 건(주식배당 주당 0.03주)
제2호 의안 : 정관 일부 변경의 건
제3호 의안 : 이사 선임의 건(사내이사 1명)
제4호 의안 : 이사 보수한도 승인의 건
제5호 의안 : 감사 보수한도 승인의 건
